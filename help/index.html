  • Help fix Arketyp for iOS 8!

  • I missed the window to update to iOS 8 and so I don't have a lot of the most updated apps. If you could hep me out, I'll love you forever!

    This may take quite a bit of time, so if you do this for me, I will include you in the beta of my next theme.

    1. Open iFile and navigation to /var/mobile/Documents. Tap Edit in the top right corner, then + in the bottom left. Add a new directory called "plists", then tap Create in the top right corner, then Done.
    2. Tap the new window icon in the bottom right corner, then the +, and choose Local Directory. Navigate to /var/mobile/Containers/Bundle/Application. If all you see are folders with seemingly random numbers and letters, tap the settings icon in the bottom left corner. Tap on File Manager and turn on the switch for Application Names, then tap Done.
    3. Now each folder should have a name. Tap the first app folder you see that isn't working properly with the theme, then tap the folder that ends in ".app". Scroll down and look for "info.plist". Once you find it, tap Edit, tap on "info.plist" to select it, tap the clipboard icon in the bottom right corner, and copy the file. Tap Done.
    4. Tap the new window icon again and select the other window. Tap the "plists" folder and paste the file in there. You will need to copy the info.plist file for each app that isn't theming properly by repeating the above steps. Because they're all named the same thing, just select Rename when it prompts you.
    5. Once you're finished with app store apps, navigate to /var/stash/_.XXXXXX/Applications (the XXXXXX are unique to your device) and copy the info.plist files for each of the system and Cydia apps that aren't themed.
    6. When you have all the info.plist files, tap Edit and then select the plists folder. Click the zip folder icon (second from bottom left, it looks like a box with a down arrow) and create the zip file. Then select the zip file and tap the email icon and email it to me at ios@gabriellew.ee.
